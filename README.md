# hypredro - A automated setup script for a **Hyprland** desktop environment

Automated setup script for a **Hyprland** desktop environment on **Arch Linux**, with pre-configured dotfiles and optional support for NVIDIA, LaTeX, CUDA, and development tools.

## 📋 Prerequisites

- Clean Arch Linux installation
- Stable internet connection
- User with sudo access (do not run as root)

## 🚩 Usage

Everything is handled by a single script:

```bash
./hypredro.sh
```

The script opens a series of dialogs **before** starting any installation, asking:

1. Whether to install **NVIDIA** drivers and hooks
2. Which **optional components** to install (checklist)

After that, it runs through the full setup without further interruption.

## 📦 Installation steps

| Step  | Action |
|-------|--------|
| 01/09 | Configure pacman (multilib, color, parallel downloads) |
| 02/09 | Install essential packages (Hyprland, Waybar, Kitty, Dolphin, etc.) |
| 03/09 | Configure NVIDIA open-dkms drivers *(if selected)* |
| 04/09 | Enable system services (NetworkManager, Bluetooth, SDDM, CUPS) |
| 05/09 | Create user directories via `xdg-user-dirs` |
| 06/09 | Copy dotfiles to `~/.config` |
| 07/09 | Install Oh My Zsh with plugins and set ZSH as default shell |
| 08/09 | Install **yay**, AUR fonts, and **Powerlevel10k** theme |
| 09/09 | Install selected optional components |

At the end, a dialog summarizes the result and asks if you want to reboot.

## 🗳️ Optional components

Selected via checklist dialog at the start:

| Component | Details |
|-----------|---------|
| LaTeX | Full TeX Live with Portuguese language support |
| CUDA | GPU computing support for NVIDIA |
| Python venv | PyTorch + NumPy, SciPy, Pandas, OpenCV, Jupyter |
| Visual Studio Code | Installed via AUR (`visual-studio-code-bin`) |
| Epson drivers | `epson-inkjet-printer-escpr` from AUR |
| OnlyOffice | `onlyoffice-bin` from AUR |
| Slack | `slack-desktop` from AUR |
| Steam | Installed via pacman |
| Discord | Installed via pacman |
| Claude Code | Anthropic CLI tool via npm |

## 👥 Included dotfiles

```
.config/
├── hypr/          # Hyprland, Hyprpaper, Hyprlock, Hypridle
├── waybar/        # Status bar
├── kitty/         # Terminal emulator
├── wofi/          # App launcher
├── dunst/         # Notifications
├── nwg-bar/       # Power menu
├── gtk-3.0/       # GTK theme
├── gtk-4.0/
├── Kvantum/       # Qt theme
├── qt5ct/
├── qt6ct/
└── Wallpapers/    # 10 wallpapers
```

## 📝 Notes

- `dialog` is installed automatically if not present — it is required to display the setup interface.
- The `.gtkrc-2.0` file is generated by `nwg-look` and contains a hardcoded path. It will be overwritten on first run of nwg-look.
- After the first login into ZSH, the Powerlevel10k wizard will start automatically. If it doesn't, run `p10k configure`.
